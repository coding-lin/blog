## 字面量类型与联合类型

```ts
interface Res {
  code: 10000 | 10001 | 50000;
  status: "success" | "failure";
  data: any;
}
```

### 字面量类型

你可能觉得很神奇，`"success"` 不是一个值吗？为什么它也可以作为类型？在 TypeScript 中，这叫做 **字面量类型（Literal Types）**，它代表着比原始类型更精确的类型，同时也是原始类型的子类型。
